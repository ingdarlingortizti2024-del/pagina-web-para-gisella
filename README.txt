PÁGINA WEB ROMÁNTICA PARA GISELLA

Cómo abrirla:
1. Descomprime el archivo ZIP.
2. Abre el archivo index.html en Google Chrome, Edge o cualquier navegador.
3. No borres la carpeta assets, porque ahí están las fotos y el video.

Música:
- La página tiene un apartado para elegir una canción desde el computador o celular.
- Para dejar una canción fija en el proyecto, coloca un archivo MP3 dentro de la carpeta assets con el nombre nuestra-cancion.mp3 y edita el archivo index.html/script.js según necesites.

Fecha del contador:
- Está configurada desde el 30/05/2026.
- Puedes cambiarla en script.js, línea: const startDate = new Date(2026, 4, 30);
